# Task Manager

A lightweight task management application built with vanilla HTML, CSS, and JavaScript, focusing on core front-end engineering fundamentals.

## Overview
This project implements a simple task manager that allows users to create, update, and remove tasks within the browser. The goal of the application is to demonstrate clean DOM manipulation, event-driven programming, and basic UI state management without relying on external frameworks.

## Key Features
- Create tasks with input validation
- Toggle task completion state
- Delete individual tasks
- Clear all tasks at once
- Keyboard support for task creation (Enter key)

## Technical Details
- Uses vanilla JavaScript for direct DOM manipulation
- Event handling implemented through both inline and programmatic listeners
- UI state managed through CSS class toggling
- Modular, readable function structure to support maintainability
